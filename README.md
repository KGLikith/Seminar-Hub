# Seminar-Hub

Seminar-Hub is a web-based application designed to manage seminar hall bookings and maintenance operations within an educational institution. The system provides a centralized platform for scheduling, approvals, and resource tracking with role-based access control.

## Features

- Seminar hall booking with real-time availability validation
- Role-based approval workflow (Teacher, HOD, Technical Staff)
- Maintenance request management for halls and equipment
- Email notifications for booking submissions, approvals, and updates
- NLP-based chatbot for availability queries and booking status checks
- Structured logging for transparency and accountability

## Technology Stack

Frontend:
- Next.js
- React
- Tailwind CSS

Authentication:
- Clerk

Backend & Database:
- Prisma ORM
- PostgreSQL
- Resend

AI & Integrations:
- OpenAI GPT-4 Mini

## Purpose

Seminar-Hub aims to improve institutional efficiency by providing a reliable, scalable, and transparent system for managing seminar hall usage and maintenance activities.
